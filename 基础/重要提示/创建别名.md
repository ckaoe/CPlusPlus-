# 命名空间别名

`namespace a = std;`

## 语法

`namespace 别名 = 命名空间名;`\
`namespace 别名 = ::命名空间名;`\
`namespace 别名 = 嵌套名::命名空间名;`


```c++
#include<iostream>
// 创建命名空间的别名
namespace a = std;

int main()
{
    a::cout << "hello world\n";
    
    return 0;
}
```


# 类型别名
`typedef int& lref;`\
`typedef int b;`

```c++
#include<iostream>
namespace a = std;
typedef int b;

int main()
{
    b c = 0;
    a::cout << c << a::endl;
    a::cout << "hello world\n";
    
    return 0;
}
```
